About interAdapt

A tool created by Aaron Fisher, Harris Jaffee, and Michael Rosenblum


interAdapt is designed to aid statisticians and clinical investigators in planning randomized trials. It can be used to determine whether certain adaptive designs offer tangible benefits compared to standard designs, in the context of investigators’ specific trial goals and constraints. Specifically, interAdapt compares the performance of trial designs with adaptive enrollment criteria to that of standard (non-adaptive) group sequential designs. Performance is evaluated in terms of power, expected trial duration, and expected sample size. Users can work interactively in the R console or with a user friendly shiny application that requires no programming experience. The shiny application contains several features for saving and re-loading the results of intermediate steps of the analysis. For example, the application allows users to save the results of the performance comparison as a tabular csv-file or as a printable, html-based report.

interAdapt is designed to work in the following open-source web browsers: Firefox or Google Chrome. However, it may also work in Safari and Internet Explorer, depending on the version.

To experiment with interAdapt, select Performance using the radio button above (which displays power curves for adaptive and standard designs). Try moving one or more sliders on the left, and then click the Apply button on the top left. After about 5 seconds, the power curves will update to reflect design performance under these new parameter settings.
